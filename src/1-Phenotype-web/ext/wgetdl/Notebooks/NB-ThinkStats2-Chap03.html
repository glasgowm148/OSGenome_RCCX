Thinks Stats: Chapter 3 - Probability Mass Function¶

Pmfs¶

A distribution is a mapping between values in the sample and their frequency.
A PMF is a mapping between values in the sample and their probability (or relative frequency).
Normalization is the process of dividing all frequencies by the sample size n.

The class size paradox¶

Assume that we have a college with the following distribution of class sizes:

 5- 9          8
10-14          8
15-19         14
20-24          4
25-29          6
30-34         12
35-39          8
40-44          3
45-49          2

This distribution can be simplified by using the average size class for each group, 5-9 has average 7, 10-14 has average 12, etc.

In [25]:
def BiasPmf(pmf, label):
    """Returns the Pmf with oversampling proportional to value.

    If pmf is the distribution of true values, the result is the
    distribution that would be seen if values are oversampled in
    proportion to their values; for example, if you ask students
    how big their classes are, large classes are oversampled in
    proportion to their size.

    Args:
      pmf: Pmf object.
      label: string label for the new Pmf.

     Returns:
       Pmf object
    """
    new_pmf = pmf.Copy(label=label)
    for x, p in pmf.Items():
        new_pmf.Mult(x, x)
    new_pmf.Normalize()
    return new_pmf
